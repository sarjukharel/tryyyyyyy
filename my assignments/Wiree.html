Wireshark Deep Dive Project

By Sarju Kharel

✅ Project Overview

This project explores network packet capture and analysis using Wireshark and PyShark. It includes a script-based tool to detect SYN flood attacks and anomalies in `.pcap` files, along with a live webpage and documentation.

🔧 Tool Scripts

  • detect_anomalies.py – Detects abnormal SYN packet counts.
  • syn_flood_detection_tool.py – Extended tool for DDoS/SYN flood detection and logging.

📂 How to Use the Tool

  1. Capture traffic with Wireshark and save it as .pcap.
  2. Run the script using Python: 
    python detect_anomalies.py
  3. For advanced detection and logs, use: 
    python syn_flood_detection_tool.py
  4. Follow command-line prompts and review printed results or generated log files.

📑 Summary

This project demonstrates the power of open-source packet capture tools and basic Python scripting to understand and detect network-based anomalies. It helped me learn to work with Wireshark, PyShark, and basic security analysis methods.
